DES   DECRYPTION

DES works by encrypting groups of 64 message bits, which is the same as 16 hexadecimal numbers. For example, if we take the ciphertext message "0000000000000000", and encrypt it with the DES key "0E329232EA6D0D73", we end up with the plaintext "8787878787878787".
